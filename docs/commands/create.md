# sandboxctl create

Create a sandbox from a profile.

## Usage

```
sandboxctl create NAME [OPTIONS]
```

## Arguments

| Name | Description | Required |
|------|-------------|----------|
| `NAME` | Sandbox name | Yes |

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--profile` | Profile to use for sandbox creation | None |
