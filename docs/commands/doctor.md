# sandboxctl doctor

Diagnose and recover sandbox issues.

## Usage

```
sandboxctl doctor NAME [OPTIONS]
```

## Arguments

| Name | Description | Required |
|------|-------------|----------|
| `NAME` | Sandbox name to diagnose. | Yes |

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--no-recover` | Skip auto-recovery, diagnose only. | `False` |

## Status

Implemented.
